O desenvolvimento de um sistema web para um grupo escoteiro tem como propósito modernizar e organizar os processos internos relacionados ao gerenciamento de jovens, atividades e informações institucionais.  O objetivo principal deste trabalho foi criar um site funcional, intuitivo e responsivo, capaz de reunir informações do grupo, registrar e atualizar dados dos jovens, controlar atividades e permitir que membros e responsáveis tenham acesso facilitado aos conteúdos essenciais do cotidiano escoteiro. Para isso, foram aplicadas tecnologias amplamente utilizadas na construção de aplicações web: HTML foi empregado para estruturar o conteúdo e garantir uma navegação clara, CSS foi utilizado para definir a identidade visual e proporcionar uma interface agradável e coerente com a proposta pedagógica do grupo, e PHP foi a base da lógica do sistema, possibilitando o gerenciamento de sessões, cadastro e atualização de dados, autenticação de usuários e interação com o banco de dados. A metodologia adotada envolveu a modelagem prévia das funcionalidades, definição de requisitos, implementação incremental e testes contínuos para assegurar o funcionamento correto das páginas e da comunicação entre front-end e back-end. Os resultados obtidos demonstram que o sistema é capaz de centralizar informações essenciais, reduzir retrabalho administrativo, facilitar o acompanhamento do progresso dos jovens e ampliar a transparência das atividades realizadas, tanto para chefes quanto para pais e responsáveis. Além disso, o uso de tecnologias simples e acessíveis garante a manutenção futura do projeto por voluntários, permitindo sua expansão conforme surgirem novas demandas do grupo. A discussão dos resultados aponta que a solução proposta contribui significativamente para a organização interna, melhora a experiência do usuário e reforça a imagem institucional do grupo escoteiro, tornando o site não apenas uma ferramenta gerencial, mas também um meio de fortalecer o vínculo entre membros, famílias e comunidade. Em síntese, o projeto demonstra que a adoção de um sistema web bem estruturado representa um avanço importante para grupos escoteiros que buscam eficiência, integração e modernização de seus processos.
Palavras-chave: desenvolvimento web; sistema de gerenciamento; escotismo; PHP; site institucional.
