Projeto para grupo escoteiro

O projeto web consiste em produzir um site para controle de membros, distintivos , entre outros para um grupo escoteiro

Aba distintivos: pode ser acessada por todos, onde o escoteiro indica os distintivos que deve receber, e seu chefe orientador confirma, com o chefe escoteiro adm podendo acessar as requisições e dar baixas

Aba jovens: cadastro simples dos jovens do grupo, com informações como ramo, idade, nome, registro UEB, orientador, entre outros

Aba escotistas: Cadastro basico dos chefes, com ramo, jovens orientados, funções, registro, entre outros

Ideias pódem ir surgindo no caminho, mas a base será essa
